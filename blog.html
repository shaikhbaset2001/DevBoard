Question-1: What are the different ways to select an element in the DOM?

You can select elements in the DOM using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), and querySelectorAll(). Each method has different use cases, with querySelector() being the most versatile.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML retrieves or sets HTML content inside an element, innerText returns only visible text without HTML, and textContent returns all text, including hidden elements, without formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element listens for events on its child elements by using event bubbling. This is useful for handling events efficiently, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and propagates upwards to its parent elements. It allows handling events at a higher level in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create an element using document.createElement(), add it using appendChild() or insertBefore(), and remove it using removeChild() or remove().
